⚓ Install

💪 Install yourself

Clone the project from GitHub and compile it like follows.

$ git clone https://github.com/tamadalab/beacher.git
$ cd beacher
$ gradle build

Then, gradle build the resultant beacher executable on ./ (root) directory.
